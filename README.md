# WxLeaveApplicationSystem

微信小程序 - 学生请假管理系统

## 功能介绍

一款面向学校/班级管理的微信小程序，提供学生信息管理和请假申请的完整解决方案。

### 核心功能

- **学生信息管理**
  - 支持导入 Excel 文件（.xlsx/.xls）批量添加学生
  - 手动添加、编辑、删除学生信息
  - 支持按姓名/学号搜索学生
  - 提供 Excel 模板下载

- **请假申请**
  - 选择学生并设置请假时间段
  - 自动计算请假时长（天/小时/分钟）
  - 时间冲突检测，防止同一学生时间段交叉
  - 支持编辑和删除请假记录

- **请假汇总**
  - 按月份查看请假统计
  - 区分全部请假记录和长期请假（>=3天）
  - 支持一键复制汇总内容

## 项目结构

```
├── pages/
│   ├── index/           # 学生信息管理页面
│   ├── student-edit/    # 学生编辑页面
│   ├── leave/           # 请假申请页面
│   ├── leave-edit/      # 请假编辑页面
│   └── summary/         # 请假汇总页面
├── utils/
│   ├── xlsx-parser.js   # Excel 解析工具
│   └── jszip.min.js     # ZIP 解压库
├── templates/
│   └── student_template.xlsx  # 学生信息导入模板
├── images/              # 图标资源
├── app.js               # 小程序入口
├── app.json             # 小程序配置
└── app.wxss             # 全局样式
```

## 技术特点

- 纯前端实现，数据存储在微信本地存储
- 使用 JSZip 解析 Excel 文件
- 支持懒加载（lazyCodeLoading）

## 使用说明

1. **导入学生信息**：在「学生信息」页面点击「从Excel导入」，选择包含学生信息的表格文件
2. **Excel 格式要求**：表格需包含「姓名」「性别」「学号」「班级」四列
3. **提交请假**：在「请假申请」页面选择学生，设置请假时间，点击提交
4. **查看汇总**：在「请假汇总」页面选择月份，查看当月请假统计
